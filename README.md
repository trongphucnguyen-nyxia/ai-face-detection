# AI Face Detection

## Overview

AI Face Detection is a computer vision application built with Python, OpenCV, and MediaPipe that automatically detects human faces in images. The program processes input images, identifies faces, draws detection bounding boxes, counts the number of detected faces, and saves annotated results for further analysis.

## Features

* Automatic face detection
* Face counting
* Bounding box annotation
* Image processing and analysis
* Runtime performance measurement
* Batch processing of multiple images
* Automatic result generation and saving

## Technologies Used

* Python
* OpenCV
* MediaPipe
* Computer Vision
* Image Processing

## How It Works

1. Images are loaded from the input directory.
2. Each image is processed using MediaPipe Face Detection.
3. Detected faces are highlighted with bounding boxes.
4. The total number of faces is calculated.
5. Processing time is measured and displayed.
6. Annotated images are saved to the output directory.
