# Java Image Processor

## Overview

Java Image Processor is a desktop application that performs image conversion, filtering, and procedural image generation through a graphical user interface.

The project demonstrates object-oriented programming principles, image manipulation techniques, and Java GUI development.

## Features

* Image conversion
* Image inversion filters
* Procedural image generation
* Wave pattern generation
* Graphical user interface
* File handling and image processing

## Technologies Used

* Java
* Java Swing
* Object-Oriented Programming

## Skills Demonstrated

* GUI application development
* Software architecture
* Image processing
* Event-driven programming
* File management
* Java development

## Purpose

This project was created to explore image manipulation techniques and desktop application development using Java.
